Introduction

Spatial Data Model

geometry + properties

shape + table

geometry = coordinates + reference properties = type + data

A GeoJSON representation

 {
      "type": "Feature",
      "geometry": {
        "type": "Point",
        "coordinates": [ 77.58270263671875, 12.963074139604124]
      },
      "properties": {
        "name": "Bengaluru"
      }
  }
